# Moonsite Spec - Claude Code Skill

Generate comprehensive technical specification documents in Moonsite format.

## What It Does

Creates detailed technical specs (איפיון) for web projects with Moonsite standards, Hebrew RTL formatting, Umbraco CMS documentation, and Figma integration.

## Installation

```bash
cp -r moonsite-spec ~/.claude/skills/
```

## Usage

Triggers when you mention "spec", "איפיון", or provide Figma designs with requirements.

**Example:** "Create a spec for the Contact page"
